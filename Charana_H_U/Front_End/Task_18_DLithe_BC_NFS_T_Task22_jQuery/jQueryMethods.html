jQuery - Get text()

text() - Sets or returns the text content of selected elements

Example 1:

            
                $(document).ready(function () {
                    $("#btn").click(function () {
                        $("#p1").text("Hello World");
                    });
                });

Results:

Text Before click: Good Morning.
